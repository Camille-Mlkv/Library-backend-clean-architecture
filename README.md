This repository demonstrates a .NET 8 backend application implemented using Clean Architecture.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/Camille-Mlkv/Library-backend-clean-architecture
   ```
2. Navigate to project directory:
   ```
   cd your_directory
   ```
3. Install dependencies:
   ```
   dotnet restore
   ```
4. Run the run_command.bat script:
   ```
   run_command.bat
   ```
5. Navigate to `http://localhost:7004/swagger` to run authentication api.
6. Navigate to `http://localhost:7005/swagger` to run core api.
